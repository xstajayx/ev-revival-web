Connect to OCPP

Once your charger is connected to Wi-Fi, you can connect it to EV Revival using the charger’s OCPP settings. This allows the charger to report status, receive commands, and use supported EV Revival features.

Before you start

  • Your Vestel charger must already be connected to Wi-Fi.
  • You need your charger ID / CBID from the side label.
  • You need access to the charger web interface login.

Set up OCPP for EV Revival

  1. Log in to the charger web interface using the same login steps shown on the Get Online page.
  2. Open the charger menu.
  3. Find and open the OCPP tab.
  4. Find Central System Address.
  5. Enter the EV Revival OCPP address in this format: wss://ocpp.jworldcreations.com/ws/CBID
  6. Replace CBID with your charger’s actual charger ID.
  7. Press Save.
  8. Restart the charger if required.
  9. Allow a few minutes for the charger to connect.

Example

Example charger ID: EU_EDR_CB12001234

Example full address:

wss://ocpp.jworldcreations.com/ws/EU_EDR_CB12001234

Important: The charger ID must match exactly. If the address is wrong, the charger will not connect. The charger also needs working Wi-Fi/internet access before OCPP can work.

If the charger does not connect, check the charger ID, Wi-Fi connection, and OCPP address carefully. A charger may fail to connect if the Wi-Fi, internet connection, firmware, DNS, routing, or backend address is incorrect.
