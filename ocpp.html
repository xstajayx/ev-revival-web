Connect Your Vestel Charger to EV Revival (OCPP)

This page is for EV Revival customers using Vestel chargers. First connect your charger to Wi-Fi on the Get Online page. Then follow the steps below to add the EV Revival OCPP address.

Before you start

  • Your charger must already be connected to Wi-Fi first.
  • You will log in using the same hotspot and login process as the Get Online page.
  • Have your charger CBID ready from the side label.

Set OCPP for EV Revival

  1. Log in to your charger web page using the same steps you used for Wi-Fi setup.
  2. Open the menu.
  3. Select OCPP.
  4. Find Central System Address.
  5. Enter: wss://ocpp.jworldcreations.com/ws/CBID
  6. Replace CBID with your own charger ID.
  7. Press Save on the OCPP page.

Important warning

Do not press Save again on the Main Page after saving OCPP.

Saving again on the Main Page can set the Central System Address back to the charger default and break EV Revival connection.

Example

Example CBID: EU_EDR_CB12001234

Example full Central System Address:

wss://ocpp.jworldcreations.com/ws/EU_EDR_CB12001234

If it does not connect first time, stay calm. Check CBID spelling carefully, confirm Wi-Fi is still working, and make sure the address is exactly correct.
