Add EV Revival to your phone

iPhone (Safari)

  1. Open EV Revival in Safari.
  2. Tap Share.
  3. Tap Add to Home Screen.
  4. Open it from your home screen icon like an app.

Android (Chrome)

  1. Open EV Revival in Chrome.
  2. Open browser menu.
  3. Tap Add to Home Screen or Install App.
  4. Launch from the icon like an app.

This is a web app experience. No app store install is required yet.
