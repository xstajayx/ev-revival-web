Get Your Charger Online

Check your charger model first. HS chargers have “HS” in the model number shown on the side label.

Before You Begin

  • Power the charger off, then back on. This reopens the charger's Wi-Fi hotspot.
  • The hotspot may only stay available for around 15 minutes after power-up.
  • Wait until the green and blue lights stop flashing before searching for the hotspot (this can take up to 2 minutes).
  • Connect to the charger's Wi-Fi hotspot using the charger serial number / CB-ID shown on the side label.
  • If your phone says Connected without internet or shows a mobile data warning, continue anyway.
  • On some Android phones, turn off mobile data and Bluetooth temporarily if the hotspot does not open correctly.
  • If the charger page opens in German, use the language dropdown at the top-right to switch language.

Typical Vestel web interface steps

  1. Open the top-left menu.
  2. Select Network Interface.
  3. Select WLAN on the left.
  4. Set WLAN to Enabled.
  5. Enter your home Wi-Fi name (SSID) and password exactly (password is case sensitive).
  6. Set Security to WPA/WPA2 PSK.
  7. Set IP Settings to DHCP.
  8. Press Save.
  9. Return to Main Page.
  10. Set Pre-Sets to WLAN, then press Save again.

A) Vestel HS chargers

Login IP:

192.168.35.1

Username: admin

Password: printed in your supplied booklet (charger-specific). Example only: F4De*5/gf. Do not assume the example password will work on your charger.

  1. Identify HS model from the side label.
  2. Connect your phone/laptop to the charger's Wi-Fi hotspot shown by the charger serial number / CB-ID.
  3. If you see a no internet warning, stay connected and continue.
  4. Open your browser and go to 192.168.35.1.
  5. Log in with username admin and your own booklet password.
  6. Open menu → Network Interface → WLAN and set WLAN to Enabled.
  7. Enter your home SSID and password exactly (password is case sensitive).
  8. Set Security = WPA/WPA2 PSK and IP Settings = DHCP, then press Save.
  9. Go to Main Page → set Pre-Sets = WLAN → press Save again.
  10. If the page does not load, disconnect mobile data/VPN and reconnect to the charger hotspot.

B) Vestel non-HS chargers

Login IP:

192.168.1.1

E.ON non-HS login: username admin / password eon01.

  1. Confirm the side label shows a non-HS model.
  2. Connect to the charger's Wi-Fi hotspot shown by the charger serial number / CB-ID.
  3. If your phone says Connected without internet, continue anyway.
  4. Open browser and go to 192.168.1.1.
  5. Log in with the correct non-HS credentials.
  6. Open menu → Network Interface → WLAN and set WLAN to Enabled.
  7. Enter your home SSID and password exactly (password is case sensitive).
  8. Set Security = WPA/WPA2 PSK and IP Settings = DHCP, then press Save.
  9. Go to Main Page → set Pre-Sets = WLAN → press Save again.
  10. If your phone keeps switching networks, temporarily disable auto-join for other Wi-Fi networks and retry.

After Saving

  • Allow up to 15 minutes for the charger to appear online.
  • When finished, close your browser and disconnect from the charger hotspot.
  • If the charger still does not come online, restart both the charger and your router, then try again.

Connectivity Tips

  • The charger relies on your own home Wi-Fi/network.
  • If the charger disconnects, commands/signals may fail.
  • This is outside the control of EV Revival.
  • If signal is weak, try a Wi-Fi extender or powerline adapter.
  • Walls, distance, garages, and weak-signal areas can all affect reliability.
